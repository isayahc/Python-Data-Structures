items are removed and added from the "top". Stacks are ordered
LIFO "Last-In-First-Out"

Stack operations:
Stack():
-creates a empty stack. No parameters needed and returns an 
empty stack
push(item):
- adds a new item to the top of the stack.
- Stack is modified
pop():
- removes the top item from the stack.
- not modified
peek():
- returns the top item of the stack but does not remove it
- stack is not modified
isEmpty():
- boolean: 0 if not empty, 1 if is
size():
- returns the number of items in the stack
